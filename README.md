# AI Investment Copilot

AI Investment Copilot is a Streamlit-based stock analysis application that helps users make better investment decisions using AI.

## Features

- 🔍 Stock Search with Suggestions
- 📈 Candlestick Chart
- 📊 RSI Indicator
- 📉 Moving Averages (MA20 & MA50)
- 🤖 AI Stock Analysis (Groq Llama 3.3 70B)
- 📰 Latest Stock News
- 💼 Portfolio Tracker
- 📊 Stock Comparison
- 🤖 AI Stock Comparison

## Technologies Used

- Python
- Streamlit
- Plotly
- yFinance
- Groq API
- Pandas
- Matplotlib

## Installation

```bash
pip install -r requirements.txt
```

Run the project:

```bash
streamlit run app.py
```
